❓ Centro de Ajuda

Dicas, sugestões e guias de uso

🚀 Como Começar

  • Clique em "Tarefas" para gerenciar suas atividades
  • Use "+ Nova Tarefa" para criar uma tarefa
  • Defina a prioridade (Baixa, Média, Alta)
  • Marque como completa quando terminar (✓)
  • Delete tarefas que não são mais necessárias

📊 Usando o Dashboard

  • Visão Geral: Métricas em tempo real
  • Análitica: Performance e produtividade
  • Usuários: Lista de usuários do sistema
  • Configurações: Personalize suas preferências

💡 Dicas & Truques

  • Use prioridades para organizar melhor
  • Filtre tarefas por status (Todas, Pendentes, Concluídas)
  • Seus dados são salvos automaticamente
  • Alterne entre tema claro e escuro (botão no topo)
  • Acesse de qualquer dispositivo - sempre sincronizado!

⌨️ Atalhos

  • Enter - Confirmar nova tarefa
  • F12 - Abrir desenvolvedor (problemas?)
  • Ctrl+K - Pesquisar (em breve)

🎨 Temas

  • Clique no botão de tema no topo da página
  • Sua preferência é salva automaticamente
  • Tema escuro (Luna) - Melhor para a noite
  • Tema claro (Solar) - Melhor para o dia

❓ FAQ

  • Dados perdidos? Verifique seu banco MySQL
  • Temas não mudam? Limpe o cache (Ctrl+Shift+Del)
  • API não responde? Reinicie o backend
  • Demora para carregar? Verifique sua conexão
